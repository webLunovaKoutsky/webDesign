Prohlášení o dostupnosti.

Tato webová stránka je vytvořena pouze na základě studentského projektu z předmětu Web Design. Stránka splňují všechny důležité zásady přístupnosti podle metodik Blind Friendly Web 2.3

Standardy a struktura webu Kód webu je vytvořen značkovacím jazykem HTML 5. Vizuální část je důsledně oddělena od obsahové části pomocí CSS (kaskádových stylů).
